# 오늘도 힘내세요! (Have a great day today!)

A premium, motivational homepage with glassmorphism effects, animated backgrounds, and interactive features.

## Features
- **Modern Design**: Glassmorphism and animated gradients.
- **Interactive**: Parallax mouse effects and a custom particle system.
- **Daily Inspiration**: Rotate through multiple motivational quotes in Korean.
- **Responsive**: Fully optimized for mobile and desktop.

## Technologies
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## How to use
Just open `index.html` in your browser.
